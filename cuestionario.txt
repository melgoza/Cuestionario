¿Cómo se organiza internet? Habla de las direcciones IP y los DNS.
EL IP es el internet protocol, que ayuda a mandar información de un dispositivo a otro, tu mandas información a una dirección, pero también mandas tu direción IP, para que ese dispositivo sepa a que dirección mandar una respuesta.
Los DNS son los que te ayudan a entrar a un sitio web, el DNS busca la dirección IP del sitio web para que tu ingreses en él.

Describe los protocolos que existen en internet, en específico HTTP y HTML
HTTP es una forma de que una computadora le pida acceso a un sitio web, tu computadora manda un "get request" a el servidor de ese sitio, lo que está pidiendo el "get request" es el código en html de ese sitio.
HTML es el lenguaje que le dice a una página como presentarse, el tamaño de letra, la fuente, si quieres poner negritas, texto qeu te enlace a otras páginas, etc.
